# Todo App

![](docs/screenshot.png)

## Start project

```sh
# clone your project from github
# Using cmd
git clone "your_git_repository"

cd react-native-todoapp-for-trainee

# install dependencies
yarn

# Running
yarn android

# or

yarn ios
```
